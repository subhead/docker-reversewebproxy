Welcome to kx1t's reverse web proxy!

This proxy uses nginx and is deployed as a docker container. If you see this page, the nginx web server is successfully installed and working. However, further configuration is required as this website is a simple placeholder.

If you want to run your own version of this reverse web proxy, please see our GitHub page.

You can download this container in a docker-enabled environment by downloading and adapting this docker-compose.yml file.
